# Rockstar Games Clone

A responsive frontend clone of the official [Rockstar Games](https://www.rockstargames.com/) website. This project replicates the modern design, layout, and interactive elements of the original site using HTML, CSS, and Bootstrap.

  - Interactive image slider with synchronized text updates.
  - Custom indicators and slide controls.

  - Grid layout displaying news cards with hover effects.
  - Responsive cards for "GTA Online", "Red Dead Online", and general Rockstar news.

- **Featured Games**:
  - Showcase of popular titles with high-quality image assets.

- **Footer**:
  - Comprehensive footer with links to corporate, privacy, and support pages.
  - Language selector dropdown.
  - Social media icons and global office locations.
